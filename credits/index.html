About Color Columns

Color Columns is a tool for exploring and copying Tailwind CSS color values. It provides an extended color palette with additional shades between Tailwind's default colors.

Features

  • Additional color shades (150, 250, etc.)
  • Copy colors in HEX, RGB, HSL, or Tailwind class format
  • Visual color grid with hover preview
  • Custom Geist font integration

Usage

Click any color to copy its value in your selected format. Hover over colors to see their shade number.
